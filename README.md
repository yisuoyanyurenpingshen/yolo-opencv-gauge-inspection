# YOLO + OpenCV 仪表盘指针状态识别系统

本项目是一个基于 YOLOv8 和 OpenCV 的仪表盘状态识别系统，主要用于从视频流中识别仪表盘指针位置，并判断当前仪表盘状态为：正常、偏高或偏低。

项目整体思路是：

1. 使用 YOLOv8 对画面中的仪表盘进行目标检测；
2. 截取仪表盘 ROI 区域；
3. 使用 OpenCV 对 ROI 内的黑色指针进行二值化和轮廓提取；
4. 根据指针方向延长采样点；
5. 使用 HSV 颜色空间判断指针所指区域；
6. 通过滑动窗口众数滤波，提升识别稳定性。

---

## 项目功能

- 支持 Windows 摄像头视频流输入
- 支持 YOLOv8 仪表盘目标检测
- 支持 OpenCV 指针提取
- 支持 HSV 颜色区域判断
- 支持滑动窗口滤波，减少单帧误判
- 支持实时可视化调试窗口
- 终端输出仪表盘状态结果

---

## 识别状态

当前系统支持以下状态判断：

| 状态 | 说明 |
|---|---|
| 正常 | 指针指向绿色区域 |
| 偏高 | 指针指向红色或橙红色区域 |
| 偏低 | 指针指向黄色区域 |
| 未知区域 | 指针采样点未落入预设颜色范围 |

终端输出示例：

```text
目前识别的是正常
仪表盘正常

目前识别的是偏高
仪表盘偏高，状态异常

目前识别的是偏低
仪表盘偏低，状态异常
